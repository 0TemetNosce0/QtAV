    • QtAV

    A media playing framework based on Qt and FFmpeg. It helps you to write a player easily. 基于Qt和FFmpeg的音视频播放框架

    Project maintained by wang-bin Hosted on GitHub Pages — Theme by mattgraham

    Video: QtAV based qmlvideofx with CedarV hardware decoding

    For Developers

    Wrtie a media player using QtAV is quite easy.

    WidgetRenderer renderer;
renderer.show();
AVPlayer player;
player.setRenderer(&renderer);
player.play("test.avi");

    For more detail to using QtAV, see the wiki Use QtAV In Your Project and examples

    QtAV can also be used in QML. QML playback API is designed to compatible with QtMultiMedia

    import QtQuick 2.0
import QtAV 1.3
Item {
    VideoOut {
        anchors.fill: parent
        source: player
    }
    AVPlayer { //or MediaPlayer
        id: player
        source: "test.mp4"
    }
    MouseArea {
        anchors.fill: parent
        onClicked: player.play()
    }
}
